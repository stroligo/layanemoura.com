## Requisitos e recomendações para o projeto web

### Conteúdo mínimo sugerido

- Hero com identidade + uma frase da bio (EN ou PT)  
- Grid de projetos com filtros: *Maps · Book Covers · Travel · Fantasy · Pattern · Commercial*  
- Página ou secção **About** (Brasil → Europa, campervan, técnicas mistas)  
- Página **Services** — capas, mapas, editorial, produto, publicidade, comissionados  
- **Contact** — `hellolayanemoura@gmail.com` + formulário opcional  
- Links para [LinkedIn](https://www.linkedin.com/in/layanemds/), [Instagram](https://www.instagram.com/layanemoura.png/), [Behance](https://www.behance.net/layanemds)  
- Galeria de alta qualidade (ilustração exige zoom e cores fiéis)

### UX / UI

- Layout que **valorize a arte** (muito whitespace, tipografia discreta, imagens grandes)  
- Suporte a **modo claro** ou fundos neutros que não competem com aquarela e tons quentes  
- Performance em mobile (audiência Instagram)  
- Acessibilidade em textos alternativos descritivos nas obras  

### Técnico

- Imagens otimizadas (WebP/AVIF) sem perder textura de papel/aquarela  
- Metadados por projeto (título, ano, ferramenta, cliente, link Behance)  
- `hreflang` ou toggle PT/EN se houver duas versões

- [ ] Idioma principal do site (EN vs PT vs bilíngue)  
- [ ] Instagram oficial: `@layanemoura.png` vs `@laydoispontos`
