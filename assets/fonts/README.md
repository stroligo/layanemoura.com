# Tipografia

Todas as fontes do site são carregadas via **Google Fonts** (`nuxt.config.ts`):

| Uso | Fonte |
| --- | --- |
| Logo / assinatura | [Delius](https://fonts.google.com/specimen/Delius) |
| Títulos | [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) |
| UI e textos | [Inter](https://fonts.google.com/specimen/Inter) |

Tokens em `src/css/tokens.css`. Não é necessário colocar ficheiros `.woff2` nesta pasta.

A assinatura no topo usa o SVG vetorial (`Projeto/layanemoura.svg` → `assets/images/logo-signature.svg`) com animação de traço em `LogoSignatureDraw` / `SiteLogo`.
